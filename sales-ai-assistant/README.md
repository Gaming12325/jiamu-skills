# Sales AI Assistant - 销售AI助手

帮助销售人员将业务场景转化为AI提示词并自动生成销售内容的Claude Code Skill。

## 功能概述

这个Skill专为销售人员设计，解决以下痛点：
- ❌ 不知道如何使用AI工具
- ❌ 不会编写有效的Prompt
- ❌ 需要快速生成高质量销售内容
- ✅ 自动识别销售场景
- ✅ 智能收集必要信息
- ✅ 生成可直接使用的内容
- ✅ 展示优化提示词供学习

**对外联络**：

3. 制定战略客户计划

## 使用方法

### 基本使用

1. **用自然语言描述你的需求**
   ```
   我要给腾讯的CTO发一封介绍我们AI客服产品的邮件
   ```

2. **Skill自动识别场景并收集信息**
   ```
   ✅ 已提取：公司=腾讯，职位=CTO，产品=AI客服
   还需要：产品价值主张
   ```

3. **补充信息**
   ```
   帮助企业实现7x24小时智能客服，降低60%人力成本
   ```

4. **获得最终结果**
   - 📝 生成的完整邮件（可直接使用）
   - 💡 优化后的提示词（供学习参考）

### 快捷调整

生成内容后，可以快速调整：
- "更正式" → 调整语气
- "简短一点" → 压缩长度
- "加数据支撑" → 增强说服力
- "换个场景" → 重新匹配

## 工作原理

### 场景识别

**智能匹配算法**（评分0-100）：
- 关键词命中：40分
- 动作词匹配：30分
- 上下文理解：30分

**处理策略**：
- ≥80分：直接推荐1个场景
- 50-79分：展示2-3个候选场景
- <50分：智能适配或引导澄清

### 信息收集

1. **智能提取**：从描述中自动识别已知信息
2. **一次性收集**：列出缺失信息，一次性询问
3. **确认调整**：展示收集到的信息，允许修改

### 内容生成

基于场景模板和用户信息，构建优化的提示词，调用AI生成最终内容。

## 安装使用

### 在Claude Code中使用

1. 将整个 `sales-ai-assistant/` 目录放入你的skills目录
2. 在Claude Code中调用：
   ```
   /sales-ai-assistant
   ```
   或直接描述销售需求，Skill会自动触发

### 自定义配置

- **修改高频场景**：编辑 `SKILL.md`，调整6个内置场景
- **添加新场景**：在对应的 `references/*.md` 文件中添加
- **调整匹配规则**：修改 `assets/scenario-keywords.json`

## 常见问题

### Q: 场景识别不准确怎么办？
A: 说"换个场景"或"不对"，Skill会重新识别或展示其他选项。

### Q: 生成的内容不满意？
A: 使用快捷指令调整（"更正式"、"简短一点"等），或提供更多信息重新生成。

### Q: 我的场景不在25个场景中？
A: Skill会智能适配最接近的场景，或基于销售最佳实践为你定制内容。

### Q: 如何学习写好提示词？
A: 每次生成后都会展示"使用的优化提示词"，仔细阅读可以学习提示词编写技巧。
