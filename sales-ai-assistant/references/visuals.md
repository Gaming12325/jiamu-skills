# 视觉素材和销售材料场景库

本文件包含视觉素材创作类别下的所有场景。这些场景利用AI的图像生成能力，帮助销售团队创建可视化内容。

---

## 场景1：在漏斗视图中可视化销售流程

**触发关键词**：销售漏斗、funnel、流程图、漏斗图、销售阶段可视化

**必填变量**：
- `[销售阶段列表]`：销售流程的各个阶段名称

**选填变量**：
- `[每个阶段的数量或比例]`：用于展示漏斗宽度

**提示词模板**：
```
创建一个销售漏斗图，展示我们的销售流程。

销售阶段：
[销售阶段列表]
（如：潜在客户 → 初步接触 → 需求确认 → 方案演示 → 商务谈判 → 成交）

阶段数据（如有）：
[每个阶段的数量或比例]

设计要求：
1. 图表简洁易读，适用于新用户入职文档或演示
2. 每个阶段清晰标注名称
3. 漏斗宽度体现各阶段的数量差异（如有数据）
4. 使用专业的配色方案
5. 可直接用于PowerPoint或培训材料

输出格式：
纯图片（或详细的文字描述供设计工具实现）
```

**使用示例**：
- 用户："画一个我们销售流程的漏斗图"
- Skill生成可视化的漏斗图描述或图片

## 场景2：可视化B2B销售漏斗

**触发关键词**：B2B漏斗、企业销售漏斗、标准销售流程

**必填变量**：
无（使用标准B2B销售阶段）

**选填变量**：
- `[自定义阶段]`：如果有特殊的销售阶段定义

**提示词模板**：
```
创建一个标准的B2B SaaS销售漏斗图。

标准阶段：
1. 寻找潜在客户（Prospecting）
2. 探索客户需求（Discovery）
3. 产品演示（Demo）
4. 提案（Proposal）
5. 成交（Closed Won/Lost）

自定义阶段（如有）：
[自定义阶段]

设计要求：
1. 使用简洁现代的图标和文字标签
2. 每个阶段配有代表性图标
3. 清晰易懂，可用于幻灯片或培训文档
4. 专业商务风格

输出格式：
清晰的漏斗图（图片或详细描述）
```

**使用示例**：
- 用户："我需要一个标准的B2B销售漏斗图用于培训"
- Skill生成标准漏斗图

## 场景3：绘制关键销售人物图

**触发关键词**：人物角色、buyer persona、客户画像、角色插图

**必填变量**：
- `[人物角色描述]`：需要绘制的角色列表及其特征

**选填变量**：
- `[使用场景]`：单页文件、培训幻灯片、营销材料等

**提示词模板**：
```
为以下销售目标人物角色创作专业插图。

人物角色：
[人物角色描述]
示例：
1. 中型企业首席财务官（CFO）- 关注成本控制和ROI
2. 全球企业IT副总裁（VP of IT）- 关注技术整合和安全
3. 物流公司运营经理 - 关注效率和流程优化

使用场景（如有）：
[使用场景]

设计要求：
1. 风格简洁现代，适合商务场景
2. 每个角色配有代表性的视觉形象
3. 可用于单页宣传资料或培训幻灯片
4. 体现角色的职业特征和关注点

输出格式：
角色插图（图片或详细描述）+ 角色标签和关键特征
```

**使用示例**：
- 用户："给我画三个目标客户角色：CTO、CFO、运营总监"
- Skill生成角色插图或描述

## 场景4：创建区域覆盖范围图

**触发关键词**：区域地图、coverage map、销售区域图、地理分布

**必填变量**：
- `[区域划分]`：需要展示的销售区域划分

**选填变量**：
- `[地理范围]`：国家、大陆、全球
- `[颜色方案]`：不同区域使用的颜色

**提示词模板**：
```
制作一张销售区域覆盖范围图。

区域划分：
[区域划分]
示例：美国按区域划分为西部、中部、东部，或中国按华东、华南、华北等

地理范围：
[地理范围]

颜色方案（如有）：
[颜色方案]

设计要求：
1. 制作简化的地图，按区域划分销售范围
2. 使用不同的颜色区域清晰区分
3. 标注关键州/省/城市
4. 简洁明了，适合用于销售启动演示文稿
5. 专业商务风格

输出格式：
区域地图（图片或详细描述）
```

**使用示例**：
- 用户："做一个中国销售区域分布图，分华东、华南、华北"
- Skill生成区域地图

## 场景5：设计一个团队庆祝图形

**触发关键词**：庆祝图形、表彰图、员工奖励、月度最佳、celebration graphic

**必填变量**：
- `[庆祝主题]`：月度最佳、季度冠军、成交庆祝等

**选填变量**：
- `[品牌风格]`：公司品牌色、风格要求
- `[包含元素]`：姓名、照片、成就等

**提示词模板**：
```
设计一张用于团队庆祝的图形。

庆祝主题：
[庆祝主题]
示例：月度最佳员工、季度销售冠军、成交里程碑庆祝

品牌风格（如有）：
[品牌风格]

包含元素：
[包含元素]
示例：需包含姓名/照片的占位符、奖杯或徽章、成就描述

设计要求：
1. 趣味十足、现代感强的视觉设计
2. 包含姓名/照片的占位符
3. 风格化的奖杯或徽章元素
4. 风格与公司内部品牌或简报保持一致
5. 可直接用于Slack、邮件或团队公告

输出格式：
庆祝图形（图片或详细设计描述）
```

**使用示例**：
- 用户："设计一个'月度销售之星'的表彰图"
- Skill生成庆祝图形设计

## 使用说明

视觉素材场景利用Claude的图像生成能力（如果可用），或提供详细的设计描述供用户使用其他设计工具实现。

**关键特点**：
- 输出通常是图片或详细的视觉设计描述
- 适用于演示文稿、培训材料、营销资料等
- 帮助销售团队快速创建专业的可视化内容

**技术说明**：
- 如果Claude支持图像生成，直接生成图片
- 如果不支持，提供详细的设计描述，用户可以使用：
  - PowerPoint/Keynote
  - Canva、Figma等设计工具
  - 外部AI图像生成工具（Midjourney、DALL-E等）

**优先级**：
当用户提到"画"、"图"、"可视化"、"设计"等关键词时，检查是否匹配这些场景。

**注意事项**：
- 视觉素材应符合商务专业标准
- 如果涉及品牌元素，提醒用户遵守品牌指南
- 提供的描述应足够详细，便于实际制作
